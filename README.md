angularjs-typescript
====================

This repository is planned as bootstrap for AngularJS projects

Installation
------------

You need grunt-cli, npm, bower and tsd in the first place.

```cd config ```

```npm install ```

```bower install ```

```tsd reinstall ```

Commands
--------
Development:

```grunt watch ```

Production:

```grunt ```
